## Overview

X-Ray Diffraction analysis project focusing on:

- **Lattice Constant Determination**: Sub-2% accuracy for LiF crystal
- **Crystal Identification**: Successfully identified unknown crystal (KBr) via structural fingerprinting
- **Bremsstrahlung Analysis**: Demonstrated Kramers theory validation
- **Planck's Constant**: Extraction from multiple voltage measurements

## Key Results

- Achieved sub-2% accuracy for LiF lattice constant determination
- Successfully identified unknown crystal structure (KBr)
- Validated Kramers theory for bremsstrahlung cutoff
- Comprehensive error analysis and uncertainty propagation

## Technical Details

- **Analysis Tools**: Python (numpy, matplotlib, scipy)
- **Data Processing**: Peak position determination, weighted averaging
- **Visualization**: Diffraction patterns and analysis plots
- **Report**: Typst document with detailed methodology

Complete experimental analysis of Hall Effect measurements in p-type and n-type Germanium semiconductors. The project involved:

- **Carrier Type Identification**: Determined carrier types via Hall voltage polarity analysis
- **Mobility Measurements**: Cross-validated mobility through Hall effect and magnetoresistance methods

- Successfully identified carrier types (n-type and p-type) via Hall voltage polarity
- Cross-validated mobility measurements using two independent methods
- Observed intrinsic transition temperature in p-type Germanium
- Comprehensive error propagation and uncertainty analysis
